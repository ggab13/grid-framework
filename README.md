## Installation 

1. - Clone the repo
 ```sh
git clone https://github.com/ggab13/grid-framework.git
   ```

2. - Navigate to the project directory
```sh
 cd grid-framework
 ```
3. - Open the index.html file in your web browser to visit the website.
